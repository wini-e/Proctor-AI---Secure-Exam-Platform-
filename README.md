**Proctor-AI** is a secure online exam platform designed to maintain the integrity of remote assessments. It monitors browser activity, detects suspicious behaviors like tab switching, and ensures students stay within a controlled exam environment.

---

## Key Features

- **Browser Monitoring:** Tracks user activity to detect any suspicious actions during an exam.
- **Tab Switch Detection:** Logs attempts to switch tabs or leave the exam window.
- **Secure Exam Environment:** Enforces full-screen mode, disables right-click, and restricts copy-paste actions.
- **Customizable Alerts:** Flags potential cheating behaviors for review.
- **Extensible Architecture:** Ready to integrate AI-based features like facial recognition or behavior analysis.

---

## Why Use Proctor-AI?

Remote exams are vulnerable to cheating without proper monitoring. Proctor-AI provides the tools to:

- Ensure fairness and integrity
- Minimize manual supervision
- Scale secure exam delivery for multiple students

It offers a balance between security and ease of use, suitable for educators, institutions, and online courses.

---

## Technology Stack

- **Frontend:** React / Next.js
- **Backend:** Node.js / Express
- **Monitoring:** Browser APIs for focus, tab, and full-screen events
- **Deployment:** Compatible with any Node.js supported hosting
